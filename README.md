# 2025 Android Application Development with Kotlin

This repository contains academic materials, source code, and practical exercises from the 2025 "Android Programming" course. It covers the full spectrum of mobile app development, from Kotlin fundamentals to advanced UI/UX design using Android Studio.

## 📱 What I Learned & Practiced

### 1. Development Environment & App Structure
- **Android Studio Configuration**: Set up the professional development environment using Android Studio.
- **Project Architecture**: Understood the basic structure of Android projects, including `AndroidManifest.xml`, `res` directories, and Gradle build scripts.

### 2. Kotlin for Android (Core Syntax)
- **Basic Syntax**: Mastered variables (`val`, `var`), data types, and null safety features.
- **Control Flow**: Implemented conditional statements (`when`, `if`) and various looping structures.
- **Advanced Kotlin**: Practiced using Classes, Inheritance, Collections, and Lambda expressions optimized for Android development.

### 3. UI/UX Design: Widgets & Resources
- **Android Widgets**: Implemented essential UI elements like `TextView`, `Button`, `EditText`, and `ImageView`.
- **Resource Management**: Managed strings, colors, and dimensions using XML resources to support maintainable code.
- **Layout Management**: Mastered various layout types including `LinearLayout`, `ConstraintLayout`, and `TableLayout` to build responsive interfaces.

### 4. Advanced Screen Composition
- **Event Handling**: Implemented ClickListeners and user interaction logic in Kotlin.
- **Complex UI Components**: Explored advanced view groups and multi-screen navigation concepts.

## 📂 Repository Contents
- **Source Code**: Practical implementation of widgets, layouts, and logic.
- **Assignments**: Weekly tasks focused on building mini-features within the Android environment.
